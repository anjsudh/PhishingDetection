Phishing Detection at Run-Time using JavaScript, DOM and AJAX

Implementation for phishing detection paper. Presented in Sai Ram Engineering College.

View project onGitHub

Uses Javascript, DOM and AJAX to detect phishing links in a mailing website.

  • JavaScript is used to perform link comparison.
  • DOM to dynamically detect all links in a message div, and set the onclick attribute to perform the comparison.
  • Ajax to automatically send the href value to the server and compare with the list of commonly known phishing sites.
